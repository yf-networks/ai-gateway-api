# AI Gateway API v0.0.1（首个公开发布版本）

AI Gateway API v0.0.1 是控制面组件的首个公开发布版本，用于统一管理 AI 网关的配置、路由与策略。本版本奠定了可扩展的、多模型 AI 流量管理基础。

## 新增（Added）

### 核心能力

- 发布 AI Gateway API 初始公开版本（控制面组件）。
- 提供 OpenAPI v1，用于管理网关策略与配置，覆盖：
	- 产品线（Products）
	- 集群/子集群（Clusters/Subclusters）
	- 实例池（Pools）
	- 域名（Domains）
	- 证书（Certificates）
	- 路由/转发规则（Routes/Forward Rules）
	- 流量调度（Traffic Scheduling）
	- 认证与鉴权（Authentication）

### 多模型路由

- 支持 AI 路由规则管理，用于灵活配置多模型路由策略。

### API Key 管理

- 支持 API Key 的关键策略能力：启用/禁用、配额、过期时间、允许模型列表、允许来源网段。

### 配置分发

- 提供导出类接口，用于将配置分发到数据面组件/Conf Agent。

### 可观测性

- 内置日志能力。
- 在监控端口暴露 Prometheus 指标（metrics）接口。
